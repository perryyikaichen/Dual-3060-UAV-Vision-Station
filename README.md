# Dual-3060-UAV-Vision-Station
Dual-RTX 3060 (24GB VRAM) AI Server for Moose V-tail UAV Autonomous Landing. Comprehensive documentation of a custom Ubuntu 24.04/Docker build, hardware-level BIOS/PCIe optimization, and NVIDIA Container Toolkit config. Bridging Computer Vision (YOLO) with ArduPilot SITL for precision vision-guided flight &amp; landing simulations.
